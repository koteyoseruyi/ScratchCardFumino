一个功能丰富的 Minecraft 刮刮卡插件，支持三种不同等级的刮刮卡、动态倍率、同类型奖励加成、全服公告等功能。

## 功能特性

- 🃏 **三种刮刮卡**：铜卡（3槽）、金卡（5槽）、钻石卡（5奖励槽+3倍率槽）
- 🎨 **精美界面**：每种卡有专属配色和布局的 GUI 界面
- 💰 **经济系统**：基于 Vault，支持所有 Vault 兼容的经济插件
- 🔄 **全局定价倍率**：通过配置可一键调整所有卡的价格和奖金
- 🏆 **同类型加成**（金卡、钻石卡）：相同奖励类型每多出现一个，该类型奖金额外增加一份基础奖金
- ✖️ **倍率系统**（钻石卡）：3个倍率槽位，乘积放大最终奖金
- 🎵 **丰富音效**：刮奖、中奖、落空、完成等不同音效
- 📢 **全服公告**：金卡和钻石卡获得超高收益时，全服广播
- 🛒 **玩家购买**：支持 `/buycard` 指令直接购买
- 🛠️ **管理指令**：支持 `/getcard` 管理员给予

## 依赖

| 依赖 | 版本 | 说明 |
|------|------|------|
| [Paper](https://papermc.io) | 1.21+ | 服务端核心 |
| [Vault](https://www.spigotmc.org/resources/vault.34315/) | 最新版 | 经济系统接口 |

## 指令

| 指令 | 权限 | 描述 |
|------|------|------|
| `/getcard <玩家> <种类> <数量>` | `scratchcard.admin` | 管理员给予玩家指定刮刮卡 |
| `/buycard <种类> <数量>` | 无（所有玩家可用） | 玩家花费金币购买刮刮卡 |

**种类可选值**：`bronze`（铜卡）、`gold`（金卡）、`diamond`（钻石卡）
